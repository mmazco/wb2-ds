# WB2 BA Demo Day DS

A responsive Next.js dashboard showcasing 16 World Build apps from the Buenos Aires Demo Day (November 15, 2025).

## Features

- 16 mini apps across 5 categories with filtering
- Utility vs Rewards scatter plot visualization
- Light/dark mode with persistence
- Fully responsive design

## Tech Stack

- Next.js 16 + React 19 + TypeScript
- Tailwind CSS
- Chart.js

## Quick Start

```bash
cd wb2-dashboard
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Categories

| Category | Apps |
|----------|------|
| Financial & Asset Management | Stack'n Finance, Earn, Valor, Rick |
| Prediction & Data Markets | Ground Truth, PumPool, TBD, Fram3s |
| Gaming, Entertainment & Education | COLONY, Squadletics, Superstream |
| Social & Identity Tools | Human Labs, Whole World Catalog, Innermost |
| Commerce & Attention | Marketplace, MADS |

## Deploy

Ready for Vercel, Railway, or any Node.js platform. No environment variables required.
